# Behavioral-Response-Analysis-System
A responsive web-based lie detector app that combines front-end dashboards, user profiles, and interactive analysis pages with a Node.js backend. The project delivers a polished experience for monitoring behavior, tracking history, and presenting authenticity insights across multiple UI views.
